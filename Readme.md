# WIFI Android connect

OS Support: **Linux only**

Connect to your device using QR code through the terminal

## run tests

```bash
cargo t
```

## run app

```bash
cargo r
```

## build NVIM plugin

```bash

chmod +x ./build.sh
./build.sh
```
